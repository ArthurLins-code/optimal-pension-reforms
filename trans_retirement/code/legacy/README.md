# Legacy Code

Scripts archived here are **no longer part of the canonical pipeline**.
They are preserved for reference and git history only.

## Why These Files Are Legacy

### F1-F7: Old counterfactual methodology
The F-stage counterfactual estimation was completely rewritten. The canonical
method is now `new_counterfactual_claiming3_pure.R` (upstream: `gabriel.R`).
F1-F7 used an older approach that has been superseded.

**Note:** The canonical scripts (`gabriel.R`, `pure.R`) still READ
`output/F/F5_table_results.csv` — an output previously generated by F5.
This output file is NOT affected by archiving the code. If regeneration
from scratch is needed, F5 must be run first to produce that CSV.

### G6: Expenditures path (abandoned)
The project reverted from the "expenditures path" to the "average benefits path"
for pure-reform computations. G6 computed DD on total expenditures (benefit x count).
The canonical G-stage is G5, which computes DD on average benefits.

### I5: Pure reforms under abandoned strategy
I5 extended I4 with pure-reform decomposition under the expenditures strategy.
Since that strategy was abandoned, I5 is no longer relevant. The canonical
I-stage is I4 (WMVPF without pure-reform decomposition).
